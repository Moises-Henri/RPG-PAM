# 🛡️ JogoRPG - Projeto (PAM)

Repositório oficial do JogoRPG. Este projeto segue a arquitetura MVC e é desenvolvido em JavaScript.

## 📖 Documentação do Projeto
Para garantir a consistência do desenvolvimento, é recomendavel que os desenvolvedores sigam as especificações:

* [📜 Regras Gerais de Combate](./docs/especificacoes_ofc.md)
* [⚡ Guia de Habilidades e Magias](./docs/sistema_habilidades.md)
* [⚖️ Tabela de Balanceamento](./docs/balanceamento.md)

## 🚀 Como Contribuir
1. Leia a documentação técnica na pasta `/docs`.
2. Certifique-se de que a lógica de negócio esteja no Model.
3. Atualize o Controller para gerenciar os novos fluxos de rodada.

## 🧭 Orientações
Para manter a organização, peço encarecidamente para os devs usar Commits Semânticos. Quando implementarem algo da documentação, usem mensagens como:

**feat: implementado cálculo de dano da Bola de Fogo (ref docs/sistema-habilidades)**

**fix: ajustado consumo de stamina do Guerreiro**
